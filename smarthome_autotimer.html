Treppenhausschaltung:

Hier muss im entsprechenden Item nur eine Zeile hinzugefügt werden.
Das Beispiel-Item sieht so aus:

Die Zeile autotimer = 5m = 0 im Item bewirkt, dass auf das Item Flur.Wandlampe oder die Gruppenadresse 0/0/12 nach einem EIN-Befehl nach 5m eine 0 gesendet bekommt und somit AUS geht.

Kurzerklärung:
  • autotimer = Aufruf/Befehl
  • 5m = gibt den Zeitraum an, nach der eine Aktion ausgeführt werden soll ( z.B. 10 = 10 Sekunden; 20m = 20Minuten )
  • 0 = Wert der zu senden ist
Es ist möglich jeden Wert für einen unterstützen Datentyp zu senden ( z.B. nach 6 Stunden einen Solltemperaturwert ).
Wichtig ist dabei, dass der zu sendende knx_dpt im Item zum Typ passen muss ( s. Grundlagen --> 1.4 Item ).
